P2736 [USACO3.4] “破锣摇滚”乐队 Raucous Rockers

题面

题目描述

你刚刚继承了流行的 “破锣摇滚” 乐队录制的尚未发表的 NNN（1≤N≤201\leq N\leq 201≤N≤20）首歌的版权。你打算从中精选一些歌曲，发行 MMM（1≤M≤201\leq M\leq 201≤M≤20）张 CD。每一张 CD 最多可以容纳 TTT（1≤T≤201\leq T\leq 201≤T≤20）分钟的音乐，一首歌不能分装在两张 CD 中。CD 数量可以用完，也可以不用完。

不巧你是一位古典音乐迷，不懂如何判定这些歌的艺术价值。于是你决定根据以下标准进行选择：

  • 1.歌曲必须按照创作的时间顺序在所有的 CD 盘上出现。(注：第 iii 张盘的最后一首的创作时间要早于第 i+1i+1i+1 张盘的第一首)

  • 2.选中的歌曲数目尽可能地多。

输入格式

第一行：三个整数：N,T,MN,T,MN,T,M。

第二行：NNN 个整数，分别表示每首歌的长度，按创作时间顺序排列。

输出格式

一个整数，表示可以装进 MMM 张 CD 盘的乐曲的最大数目。

样例 #1

样例输入 #1

4 5 2
4 3 4 2

样例输出 #1

3

提示

题目翻译来自NOCOW。

USACO Training Section 3.4

思路

对于这道题目，朴素做法必须要存储所有专辑的已用长度，但是你要注意一件事情，唱片的时间必须是有序的，那么如果第 iii 个音乐不放在 jjj 个专辑，那么所有后面的音乐都会放在 j+1j+1j+1 以后的专辑. 那么我们可以设 dfs(i,l,n,s)dfs(i,l,n,s)dfs(i,l,n,s) 表示在放 iii 个唱片的时候第 nnn 个唱片已经放了 lll 分钟的歌且 所有专辑里面一共有 sss 个音乐，那么这个代码就很好谢了.

代码

#include<iostream>
using namespace std;
int N,T,M,A[21],ans=-1;

void dfs (int i,int l,int n,int s) {
	if (i > N || n > M){
		ans=max(ans,s);
		return ;
	}
	if (T - l >= A[i]) {
		if (T - l > A[i])dfs(i + 1,l + A[i],n,s + 1);/*如果这个歌曲还可以放在当前的专辑里面且还有剩余时间不用换成新的唱片*/
		dfs (i + 1,0,n + 1,s + 1);
	}
	dfs (i + 1,l,n,s);//不录歌曲
}
int main(){
	cin>>N>>T>>M;
	for(int i=1;i<=N;i++)cin>>A[i];
	dfs(1,0,1,0);
	cout<<ans;
}
