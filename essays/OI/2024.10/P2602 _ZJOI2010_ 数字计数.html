P2602 [ZJOI2010] 数字计数

题面

题目描述

给定两个正整数 aaa 和 bbb，求在 [a,b][a,b][a,b] 中的所有整数中，每个数码(digit)各出现了多少次。

输入格式

仅包含一行两个整数 a,ba,ba,b，含义如上所述。

输出格式

包含一行十个整数，分别表示 0∼90\sim 90∼9 在 [a,b][a,b][a,b] 中出现了多少次。

样例 #1

样例输入 #1

1 99

样例输出 #1

9 20 20 20 20 20 20 20 20 20

数据规模与约定

  • 对于 30%30\%30% 的数据，保证 a≤b≤106a\le b\le10^6a≤b≤10​6​​；
  • 对于 100%100\%100% 的数据，保证 1≤a≤b≤10121\le a\le b\le 10^{12}1≤a≤b≤10​12​​。

#include<iostream>
#include<vector>
using namespace std;
typedef long long ll;
ll sum[10],ed,srt;
void dfs(ll x){
	if(x<srt)return;
	dfs(x-1);
	while(x){
		sum[x%10] ++ ;
		x /= 10;
	}
} 
int main(){
	cin>>srt>>ed;
	dfs(ed);
	for(int i=0;i<10;i++)cout<<sum[i]<<' ';
}
